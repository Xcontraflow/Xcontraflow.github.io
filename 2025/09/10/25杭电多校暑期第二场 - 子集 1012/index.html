25杭电多校暑期第二场 - 子集 1012

Author: contraflow Date: September 10, 2025  9:23:25 Category: 杭电多校

题面

Problem Description

请注意本题特殊的空间限制。（2MB）
给出一个长度为 $n$ 的非负整数序列 $a$，你可以在这些数中选取任意个数（可以是零个），但不能选取相邻的数，求选出来的数的异或和最大值。

Input

每个测试点中包含多组测试数据。输入的第一行包含一个正整数 $T \ (1 \leq T \leq 3)$，表示数据组数。对于每组测试数据：
第一行一个正整数 $n \ (1 \leq n \leq 50)$，表示序列长度。
第二行 $n$ 个非负整数 $a_1, a_2, \dots, a_n \ (0 \leq a_i \leq 10^{18})$，表示序列 $a$。

Output

对于每组测试数据：输出一行一个整数，表示答案。

[!Tip]- 官方题解
考虑从左往右，贪心地将数字插入线性基中。
首先 $a_1, a_2$ 必须要选其中一个插入线性基。由于多插入一个数字，一定不会使得线性基能表示的集合减少，因此插入的相邻两个数的位置间隔必为 $1$ 或 $2$（若间隔超过 $2$，将中间的数字插入线性基一定不劣）。

这样做会算重，但本题只是求异或最大值，故没有影响。

设 $g(i)$ 表示：考虑到了 $a_1 \sim a_i$，且 $a_i$ 必须被插入时，这样的线性基个数。则有 $g(i) = g(i-2) + g(i-3)$

直接进行搜索，要枚举的线性基个数为 $g(n) + g(n-1)$
当 $n = 50$ 时，$g(n) + g(n-1) = 1221537$，可以接受。

思路

就我目前的理解来看，线性基是一个 用来承载多个元素信息的异或线性独立的数集

有以下3个性质：

  • 原序列的任意一个数都可以由线性基内部的一些数异或得到（承载信息）
  • 线性基内部的任意数异或起来都不能得到 0（线性独立）
  • 线性基内部的数个数唯一；且在保持性质一的前提下，数的个数是最少的（最优结构）

每次对线性基插入元素的时候，都从最高位往地位找，看这个最高位是否存在，能否被已有的基消掉，不能就加入线性基中

要求一个数集的最大异或和，也是用这个数基的线性基，将其从高位到地位异或，贪心的找这个基能否让结果变大即可
（贪心证明可见后记）

这道题是间隔一个进行异或，那么很显然第一个或者第二个数一定在基中，所以main函数中把他们作为开始

为什么 DFS 只跳 +2 或 +3 就够了？

[!success]+ 来自GPT的解释
这道题的难点在“不能选相邻”，如果直接枚举子集是指数级。但题解用到了一个关键观察（PPT 上那句）：
只求“最大异或值”，往线性基里多插入一些数，不会让最大值变小（空间只会更大或不变）
因此，两次插入的位置间隔如果超过 $2$，插入中间某个数不会变差。

换句话说：在“合法子集”（两两不相邻）里，假如你选了下标 i 和 j (j ≥ i+3)，那完全可以把中间 i+1（甚至 i+2）也“插入到基里”。
虽然这在“选子集”意义上会破坏不相邻约束，但我们不是在枚举子集，我们是在构造一个能产生至少同样大最大异或值的线性基——而多放向量“不吃亏”。

于是我们可以只枚举“相邻选点间隔为 $2$ 或 $3$”的序列（即选择 i 后，下一个只需要在 i+2 或 i+3 处“放一个数进基”），就不会漏掉最优解：

  • 原本最优解对应的某个合法子集，若有更大间隔 $(\geq 3)$，我们能“补点”把间隔压到 $2$ 或 $3$，线性基的最大值不变或更大；
  • 因此一定存在一个“间隔均为 $2$ 或 $3$”的插入序列，能达到全局最优的最大 XOR；
  • 我们枚举所有这类序列，取 $\max$ 即可。
