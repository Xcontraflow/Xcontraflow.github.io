25杭电多校暑期第五场 - k-MEX 1007

Author: contraflow Date: September 12, 2025  7:52:34 Category: 杭电多校

题面

Problem Description

给出两个正整数 $n, k$。你需要计算：从 $0, 1, \dots, n-1$ 这 $n$ 个整数中随机选择 $k$ 个不同的整数，组成的集合的 $\mathrm{MEX}$ 的期望是多少？
可以证明答案为一个有理分数 $\frac{x}{y}$，你需要输出这个分数对 $10^9 + 7$ 取模后的结果。

$\mathrm{MEX}$ 表示集合内最小的未出现的自然数，例如：

  • $\mathrm{MEX}({1, 0, 2, 4}) = 3$
  • $\mathrm{MEX}({1, 2, 3, 4}) = 0$

Input

  • 第一行一个正整数 $T \ (1 \leq T \leq 10^4 + 7)$，表示数据组数。
  • 对于每一组数据，输入两个正整数 $n, k \ (1 \leq k \leq n \leq 10^9)$。

Output

对于每一组数据，输出一行一个整数，表示期望 $\mathrm{MEX}$ 对 $10^9 + 7$ 取模后的结果。

[!Tip]- 官方题解

思路

对于 $\sum_{i=1}^ki\cdot P(i)=P(1)+(P(2)+P(2))+(P(3)+P(3))+\cdots+(P(k)+\cdots+P(k))$

可以写成：$\sum_{i=1}^ki\cdot P(i)=[P(1)+P(2)+\cdots+P(k)]+[P(2)+\cdots+P(k)] + \cdots + [P(k)]$

而 $[P(i)+P(i+1)+\cdots+P(k)]$ 其实就是 $\Pr[MEX\geq i]$
（有 $\Pr[MEX = i] = P(i)$）

所以 $\sum_{i=1}^ki\cdot P(i)=\sum_{i=1}^k\Pr[MEX\geq i]$

而 $\Pr[MEX\geq i]$ 即为 $MEX \geq i$ 的情况除以 所有的方案数

所以，所求即为：
$$
\sum_{i=1}^k\frac{\binom{n-i}{k-i}}{\binom nk}
$$

经过题解里的化简可以得到最后的结果为 $\frac k{n-k+1}$

化简部分用到的原理本质是 杨辉三角恒等式：
$$
\binom rr+\binom{r+1}r+\binom{r+2}r+\cdots+\binom nr=\binom{n+1}{r+1}
$$
对应的一步一步，是 帕斯卡恒等式：
$$
\binom ab+\binom a{b+1}=\binom{a+1}{b+1}
$$
