25杭电多校暑期第三场 - 三带一 1004

Author: contraflow Date: September 9, 2025  21:29:22 Category: 杭电多校

题面

Problem Description

一副扑克牌有 $a_1$ 张 $A$, $a_2$ 张 $2 \dots a_{10}$ 张 $10$, $a_{11}$ 张 $J$, $a_{12}$ 张 $Q$, $a_{13}$ 张 $K$，我们希望打出尽可能多的三带一，三带一指的是打出的牌需要是 AAA B 类型，即四张牌中三张牌需要是相同的，剩下的一张牌必须和前面的牌不同。
例如 $AAA2, 3338, JJJ9$ 这些是三带一，而 $6666, 6677, 6779, 6789$ 都不是三带一。
使用过的牌不能重复使用。

Input

输入包含 $T$ 组样例；
输入第一行有一个整数 $1 \leq T \leq 2 \times 10^4$，表示样例组数，每组格式如下：
每行有 $13$ 个整数表示组 $a$，即 $a_1, a_2, …, a_{13}$ ($0 \leq a_i \leq 10^6$)。

Output

输出包含 $T$ 行，对于每一组样例你都需要输出一个答案代表最多能打出多少个三带一。

[!Tip]- 官方题解
二分答案，判断某 Ans 是否有解：
由于 $c_i$ 不能负有限制：

一、$\forall i \rightarrow b_i \leq \left\lfloor \frac{a_i}{3} \right\rfloor$
则有 $\sum a_i = A, \sum b_i = B = Ans, \sum c_i = C = A - 3Ans$；
由于每个 “3” 都需要匹配一个 “1”，有限制：

二、$B\leq C\to Ans\leq\lfloor\frac A4\rfloor$
另外，给定所有 $b_i$ 和 $c_i$，存在方案使每个 “3” 匹配上不同数的充要条件是 $max(b_i + c_i) \leq C$：

  • 当存在某 i 使得 $b_i > C - c_i$ 时，显然该 i 的 $b_i$ 个 “3” 无法被匹配；
  • 否则存在某 i 使得 $b_i + c_i = C$ 时，匹配的策略显然；
  • 否则由于任意一次操作仅使 $C$ 减小，且各 i 的 $b_i + c_i$ 不升，可能意图匹配直至出现 $max(b_i + c_i) = C$。

即 $\forall\mathrm{~}i\to b_i+c_i\leq C$；又 $b_i + c_i = a_i - 2b_i$，得限制：

三、$\forall i \rightarrow b_i \geq \left\lceil \frac{a_i - A + 3Ans}{2} \right\rceil$
整理有限制，得 $[Ans\leq\frac A4]\wedge[\mathrm{~}\forall i,\lceil\frac{a_i-A+3Ans}2\rceil\leq b_i\leq\lfloor\frac{a_i}3\rfloor]$，同时对 $b_i$ 的上下限判断Ans的合法性（注意单个 $b_i$ 的上界之间也不能冲突）

思路

下面主要是对题解的解读

主要是数学推导，其中：

  • $a_i$ 代表每种牌的数量，$A$ 代表牌的总数
  • $b_i$ 代表每种牌能抽出的“3”的数量，$B$ 代表总共能抽出“3”的数量
  • $c_i$ 代表每种牌抽出“3”后剩余的数量，$C$ 代表剩余的总牌数

很简单可以得知 $C = A - 3Ans$、$c_i=a_i-3b_i$，且 $B= Ans$

$B\leq C$ 的原因是，$B$ 代表的是“3”的数量，$C$ 代表的是“剩余牌”的数量，那么显然之间有个配对关系，即剩余牌可以有多余

将上面的第一行移项后容易得到 $Ans\leq\lfloor\frac A4\rfloor$

同样的，因为“三带一”的“一”，必须要是和自己的是不一样的，所以必须是从别的堆里选，可以得到 $b_i \leq C - c_i$，易得 $\max(b_i+c_i)\leq C$

将 $c_i=a_i-3b_i$ 带入，可以得到 $b_i+(a_i-3b_i)\leq C$，化简移项后得到 $b_i\geq\frac{a_i-C}2$

当得到一个下界不等式的时候，应该将 $b_i$ 向上取整：$b_i\geq\left\lceil\frac{a_i-C}2\right\rceil$，最后得：$b_i\geq\left\lceil\frac{a_i-A+3Ans}2\right\rceil$

整理得：
$$
[Ans\leq\frac A4]\wedge[\mathrm{~}\forall i,\lceil\frac{a_i-A+3Ans}2\rceil\leq b_i\leq\lfloor\frac{a_i}3\rfloor]
$$

那么，接下来只需要遍历 $b_i$，去判断是否在上下界内即可

如何判断呢？——二分答案
对 $0 \sim \frac A4$ 进行二分即可

  • L 指向第一个不可行的位置（first false）
  • R = L - 1 指向最后一个可行的位置（last true）
    输出 R
