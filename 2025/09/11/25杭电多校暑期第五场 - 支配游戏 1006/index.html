25杭电多校暑期第五场 - 支配游戏 1006

Author: contraflow Date: September 11, 2025  18:49:36 Category: 杭电多校

题面

Problem Description

给定一个由若干连通块组成的无向图，任意两个节点间最多只有一条边。每个连通块是简单路径或者简单环。
两个玩家交替在图上选点博弈。每一轮，对应玩家选取一个能够支配至少一个未被支配节点的节点，并将该节点和所有被其支配的节点标记为已支配。
（称一个节点 $u$ 支配另一个节点 $v$ 当且仅当 $u$ 和 $v$ 相邻）
若当前玩家无法选取节点，则该玩家输掉游戏。现在需要判定先手是否必胜。

Input

第一行包含一个整数 $t \ (1 \leq t \leq 10^4)$，表示测试数据的组数。
每组测试数据的第一行包含一个整数 $n \ (1 \leq n \leq 100)$，表示该图中连通块的数量。

接下来的 $n$ 行，每行描述一个连通块，包含两个整数：

  • $num$ $(2 \leq num \leq 10^{18})$：该连通块中的顶点数量；
  • $is_path \in {0, 1}$：表示该连通块的类型。

若 is_path = 1，该连通块是一条由 num 个点组成的 简单路径。
若 is_path = 0，该连通块是一个由 num 个点组成的 简单环，其中 $num \geq 3$。

Output

对于每组测试数据，若先手玩家有必胜策略，则输出一行 Yes，否则输出一行 No。

[!Tip]- 官方题解
该问题为公平博弈问题，所以考虑采用 SG 定理来求解问题

令 $P_{i,n} \ (n \geq 2, i \in {0,1,2})$ 为长度为 $n$，左右端点中的 $i$ 个端点被支配且其余节点均未被支配的路径，$C_n \ (n \geq 3)$ 为长度为 $n$ 且所有节点均未被支配的环，则通过打表和归纳法可得所有这些图的 SG 值如下所示。

  • 当 $i = 2$ 时。对于 $n \leq 2$，$SG(P_{2,n}) = 0$；对于 $n > 2$ 有
    $$
    SG(P_{2,n}) = \operatorname{mex}{i=1}^{n} \big( SG(P{2,i-1}) \oplus SG(P_{2,n-i}) \big) =
    \begin{cases}
    2 & n \bmod 4 = 0 \
    3 & n \bmod 4 = 1 \
    0 & n \bmod 4 = 2 \
    1 & n \bmod 4 = 3
    \end{cases}
    $$
  • 当 $i = 1$ 时。对于 $n \leq 1$，$SG(P_{1,n}) = 0$；对于 $n > 1$ 有
    $$
    SG(P_{1,n}) = \operatorname{mex}{i=1}^{n} \big( SG(P{2,i-1}) \oplus SG(P_{1,n-i}) \big) =
    \begin{cases}
    3 & n \bmod 4 = 0 \
    0 & n \bmod 4 = 1 \
    1 & n \bmod 4 = 2 \
    2 & n \bmod 4 = 3
    \end{cases}
    $$
  • 当 $i = 0$ 时。对于 $n = 3$，$SG(P_{0,n}) = 2$；对于 $n \neq 3$ 有
    $$
    SG(P_{0,n}) = \operatorname{mex}{i=1}^{n} \big( SG(P{1,i-1}) \oplus SG(P_{1,n-i}) \big) =
    \begin{cases}
    0 & n \bmod 4 = 0 \
    1 & n \bmod 4 = 1 \
    1 & n \bmod 4 = 2 \
    3 & n \bmod 4 = 3
    \end{cases}
    $$
  • 对于 $n \geq 3$ 有
    $$
    SG(C_n) = \operatorname{mex}(SG(P_{2,n-1})) =
    \begin{cases}
    1 & n \bmod 4 = 3 \
    0 & \text{otherwise}
    \end{cases}
    $$
    最后把所有连通块的 SG 值异或起来即可得到整个图的 SG 值。若 SG 值为 $0$，则先手必败；否则先手必胜。

思路

你这道题描述的是：

  • 图被拆成若干不相交的连通块（要么是简单路径，要么是简单环）
  • 每一步操作是“选择一个点，并支配至少一个未支配点”
  • 游戏在所有点都被支配后结束，不能操作的人输
    这正好是一个公平博弈问题：
  • 每个连通块就是一个独立的子游戏。
  • 整个游戏的局面就是这些子游戏的“和”
    所以:
  • 我们需要给每个子游戏（路径or环）求出它的SG值。
  • 最后把所有连通块的SG值做异或和。
  • 如果结果=0，说明先手必败；否则先手必胜。

SG函数

所有 Sprague–Grundy 定理里的 SG 函数，都按照这一定义：
$$
SG(x)=mex{SG(y)\mid y\text{ 是从状态 }x\text{ 走一步能到达的状态 }}
$$

$x$ 为当前局面，$y$ 为 $x$ 走一步之后的后继状态
$mex$（minimal excludant）：集合里没有出现过的最小非负整数

[!Tip]- 例子

Nim 游戏，石子堆大小为 $n$
  • 能走到的状态就是拿掉若干石子后的堆：$n-1, n-2, \dots, 0$。
  • 所以：$SG(n) = \operatorname{mex}{ SG(0), SG(1), \dots, SG(n-1) }$
  • 递推下去会发现 $SG(n) = n$
取石子游戏：每次只能取 1 或 2 个
  • 状态 $n$：堆里有 $n$ 个石子。
  • 可走：到 $n-1$ 或 $n-2$。
  • 所以：$SG(n) = \operatorname{mex}{ SG(n-1), SG(n-2) }$
  • 算出来的 SG 序列是周期性的：$0, 1, 2, 0, 1, 2, \dots$

为什么一次落子会变成“两个子游戏的异或”

在 $P_{2,n}$ 里你能下的只有 中间的尚未被支配的点，假设你选择了从左数第 $i$ 个点：

  • 这一手把第 $i$ 个点“选中/支配”。根据支配博弈的语义，这个被选中的点会把它 相邻的点也一并标记已支配（它们之后都不可再选）。
  • 于是整条路径被“切”成 左右两个互不干扰的段：左侧包含原来的前 $i-1$ 个点，右侧包含后 $n-i$ 个点。
  • 更关键的是：这两个段的 两端现在都已被支配：
    • 左段的左端本来就被支配；左段靠近切口的那端因为邻着被选中的点，也被支配；
    • 右段同理。
  • 所以左右两段都变成了 同类状态：$P_{2,i-1}$ 和 $P_{2,n-i}$

独立子游戏的 SG 值相加用的是 Nim 和，也就是异或：$SG = SG(P_{2,i-1}) \oplus SG(P_{2,n-i})$

为什么要对所有 $i$ 取 $\mathrm{mex}$

SG 的定义是：**当前局面的 SG 值 = 你所有可走一步之后到达局面的 SG 值集合的 $\mathrm{mex}$**。

可走的一步就是“在某个位置 $i$ 落子”，所以把所有可能的 $i$ 产生的结果（也就是上面那个异或）收集起来，取 $\mathrm{mex}$，就得到当前 $P_{2,n}$ 的 SG 值。
（严格一点，能落的位置其实是 内点（端点已被支配，不能选），所以严格应是 $i \in [2, n-1]$）

演化过程：

我们以路径、且左右两边被支配为例：
$$
SG(P_{2,n}) = \operatorname{mex}{ SG(P_{2,i-1}) \oplus SG(P_{2,n-i}) \mid 1 \leq i \leq n }.
$$

初始条件：

  • $SG(P_{2,0}) = 0$ （空路径，必败）
  • $SG(P_{2,1}) = 0$ （一个点且是端点 $\Rightarrow$ 已支配，不能再走）
  • $SG(P_{2,2}) = 0$ （两个端点已支配 $\Rightarrow$ 没有剩余点）

计算：

  • $n=3$

    • 可选点只有中间 1 个，左右两边都是 $P_{2,1}$，$SG=0$
    • 所以集合 = ${0 \oplus 0} = {0}$
    • $\operatorname{mex}{0} = 1$
    • $\Rightarrow SG(3) = 1$

  • $n=4$

    • 可能的切法：
      • $i=2 \Rightarrow (P_{2,1} \oplus P_{2,2}) = 0 \oplus 0 = 0$
      • $i=3 \Rightarrow (P_{2,2} \oplus P_{2,1}) = 0 \oplus 0 = 0$
    • 集合 = ${0}$
    • $\operatorname{mex}{0} = 1$
    • $\Rightarrow SG(4) = 1$

  • $n=5$

    • $i=2 \Rightarrow (P_{2,1} \oplus P_{2,3}) = 0 \oplus 1 = 1$
    • $i=3 \Rightarrow (P_{2,2} \oplus P_{2,2}) = 0 \oplus 0 = 0$
    • $i=4 \Rightarrow (P_{2,3} \oplus P_{2,1}) = 1 \oplus 0 = 1$
    • 集合 = ${0,1}$
    • $\operatorname{mex}{0,1} = 2$
    • $\Rightarrow SG(5) = 2$

  • $n=6$

    • 类似操作，得到集合 = ${1,2}$
    • $\operatorname{mex}{1,2} = 0$
    • $\Rightarrow SG(6) = 0$

  • $n=7$

    • 集合 = ${0,2}$
    • $\operatorname{mex}{0,2} = 1$
    • $\Rightarrow SG(7) = 1$
      你会发现：结果开始循环了

将循环归纳，可以得到，对于 $n>2$，有：
$$
\mathrm{SG}(P_{2,n})=\mathrm{mex}{i=1}^n(\mathrm{SG}(P{2,i-1})\oplus\mathrm{SG}(P_{2,n-i}))=\begin{cases}2&n\mathrm{mod}4=0\3&n\mathrm{mod}4=1\0&n\mathrm{mod}4=2\1&n\mathrm{mod}4=3&\end{cases}
$$
其余的结论也是通过找到周期之后归纳得来

后记

路径：一系列顶点 $v_1, v_2, \dots, v_k$，相邻顶点之间都有边相连
简单路径：没有重复顶点的路径（直线型）
简单环：一个闭合路径（首尾相同），除了起点和终点外，不允许有重复的顶点或边

题目来源：2025“钉耙编程”中国大学生算法设计暑期联赛（5） 1006

[!note]- 完整版题面
两个玩家在一张无向图上进行博弈，任意两个节点之间最多一条边相连。这张图由若干个 互不相交 的连通块组成，每个连通块要么是一条 简单路径，要么是一个 简单环
在游戏中，两个玩家轮流行动。在每一轮中，当前玩家必须选择一个顶点 $u$，该顶点需 能支配 至少一个 未被支配 的顶点。注意，该顶点本身并不要求是未被支配的。一个顶点 $u$ 能够支配它自己以及所有与它相邻的顶点。
当某个顶点被选中后，它以及所有被其支配的顶点都会被标记为 已支配。
若某位玩家在其回合无法进行操作（即没有可以支配未被支配顶点的顶点可选），则该玩家输掉游戏。

请判断，在双方都采取最优策略的前提下，先手玩家是否必胜。
