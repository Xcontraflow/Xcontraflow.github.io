25杭电多校暑期第五场 - 信赖跃迁：置换轨迹计算 1004

Author: contraflow Date: September 11, 2025  12:12:10 Category: 杭电多校

题面

（原题面的 Problem Description 实在太恶心了，这道题一大难点就是题目看不懂，所以这里直接用了题解里的简化版）

Problem Description

给定长度为 $n$ 的置换 $q$ 和一个长度为 $m$ 的整数序列 $k_1, k_2, \dots, k_m$，计算存在多少个长度为 $m+1$ 的置换序列 $p_0, p_1, \dots, p_m$ 能够满足：$p_0 \circ p_1^{k_1} \circ p_2^{k_2} \circ \cdots \circ p_m^{k_m} = q$

其中 $\circ$ 是置换的复合运算。结果对 $10^9 + 7$ 取模。

Input

第一行包含一个整数 $t \ (1 \leq t \leq 10)$，表示数据组数。每组数据输入如下：
第一行包含两个整数 $n \ (1 \leq n \leq 10^5)$ 和 $m \ (1 \leq m \leq 10^5)$，表示置换 $q$ 的长度和序列 $k$ 的长度。
第二行包含 $n$ 个整数 $q_0, q_1, \dots, q_{n-1}$，表示置换 $q$。保证 $q$ 是一个有效的置换，即每个元素都不重复且在 $[0, n-1]$ 之间。
第三行包含 $m$ 个整数 $k_1, \dots, k_m \ (1 \leq k_i \leq 10^5)$，表示序列 $k$。

Output

对于每组数据，输出一行一个整数，表示满足条件的置换序列 $p$ 的个数。由于答案可能非常大，请输出答案对 $10^9 + 7$ 取模的结果。

[!Tip]- 官方题解
无论 $p_1, \dots, p_m$ 是何种置换序列，$p_0$ 都可以唯一确定地将置换复合结果变成 $q$。因此答案为：
$((n!)^m) \bmod (10^9 + 7)$

思路

群论，第一次碰到群论的题

先来理解 置换 这个概念：
长度为 $n$ 的置换，其实就是把 ${0, 1, 2, \dots, n-1}$ 里的数重新排一遍 ，有点像全排列
比如 $n = 3$，所有的置换有 $3! = 6$ 种：
$[0, 1, 2]$ （恒等置换，啥也没变）、$[0, 2, 1]$、$[1, 0, 2]$、$[1, 2, 0]$、$[2, 0, 1]$、$[2, 1, 0]$

置换的复合 (composition) 就是 “先做一个置换，再做另一个置换”，类似与乘法
例如：

  • $p = [1, 0, 2]$ 表示 $0 \to 1, \ 1 \to 0, \ 2 \to 2$；
  • $q = [2, 1, 0]$ 表示 $0 \to 2, \ 1 \to 1, \ 2 \to 0$；
    那么 $p \circ q$ 就是 “先 $q$ 后 $p$”：
  • $0$ 经 $q$ 变成 $2$，再经 $p$ 变成 $2 \to 2$，所以 $0 \to 2$；
  • $1$ 经 $q$ 变成 $1$，再经 $p$ 变成 $0$，所以 $1 \to 0$；
  • $2$ 经 $q$ 变成 $0$，再经 $p$ 变成 $1$，所以 $2 \to 1$。
    因此 $p \circ q = [2, 0, 1]$。

题目要求，$p_0\circ p_1^{k_1}\circ p_2^{k_2}\circ\cdots\circ p_m^{k_m}=q$
我们可以随便 $p_1$ 到 $p_m$ ，但是一旦后面的这些都确定了，前面的 $p_0$ 也确定了

我们令：$g=p_1^{k_1}\circ p_2^{k_2}\circ\cdots\circ p_m^{k_m}$
有 $p_0=q\circ g^{-1}$

那就看 $(p_1, p_2, \dots, p_m)$ 能有多少种可能。

  • 每一个 $p_i \ (i \geq 1)$ 都是一个 长度为 $n$ 的置换。
  • 长度为 $n$ 的置换有多少个？答案是 $n!$ （因为是 $n$ 个数的全排列）。
  • 所以 $p_1$ 有 $n!$ 种选法，$p_2$ 也有 $n!$ 种选法，……，$p_m$ 也有 $n!$ 种选法。
    这些选择互相独立，因此： $\text{总数} = (n!) \times (n!) \times \cdots \times (n!) \quad$（一共 m 个因子），也就是：$(n!)^m$

直接用快速幂计算即可

总计难度：中位绿~下位蓝
