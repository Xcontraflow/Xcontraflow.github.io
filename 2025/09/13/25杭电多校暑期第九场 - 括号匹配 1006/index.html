25杭电多校暑期第九场 - 括号匹配 1006

Author: contraflow Date: September 13, 2025  19:16:11 Category: 杭电多校

题面

Problem Description

本题中字符串下标从 1 开始。
对于字符串 $S$，我们称 $S$ 的空子串为其中连续一段字符构成的字符串。
第 $l$ 个字符到第 $r$ 个字符 $(l \leq r)$ 构成的子串记作 $S[l \dots r]$。
称非空字符串 $S$ 是回文串，当且仅当所有字符的顺序翻转过来后 $S$ 保持不变。
例如 () 不是回文串，而 ()( ) 是回文串。
称非空字符串 $S$ 是括号串，当且仅当 $S$ 中只有 ( 和 ) 两种字符，且数量相等，
并且对于任意的 $i$，$S[1 \dots i]$ 中 ( 的数量不少于 )。
例如 () 是括号串，而 ())( 不是括号串。

给出一个字符串 $S$，求 $S$ 有多少个子串 $T$ 同时满足：

  • $T$ 是回文串；
  • $T$ 在某一个子串是括号串。
    两个子串的下标范围不同，我们认为它们是两个不同的子串。

Input

输入一行一个字符串 $S$ ($1 \leq |S| \leq 10^4$)，
$S$ 中仅含小写字母和小括号，含义同题目描述。

Output

输出一行一个自然数，表示答案。

[!Tip]- 官方题解
注意到“存在一个子串是括号串”当且仅当存在一个子串是 ()。

分类讨论奇偶回文串。以奇回文串为例：
枚举每个对称中心 $c$，尝试从半径 $r=0$ 开始向外扩展，
并同时维护“是否包含 ()”以及“当前是否回文”两个 tag。

对于所有合法的 $r$，让答案增加 1 即可。

思路

所谓的“存在一个子串是括号串”，其实就是一定包含 “（）”
那么只需要先遍历，找所有括号的位置，打上标记：f 数组
之后先遍历奇中心同时向外拓展，不回文了就跳出
因为新产生括号一定会在两边产生，所以对两边进行或运算

偶中心同理，注意索引和检查越界

时间复杂度：$O(n^2)$

总体难度：中位黄
